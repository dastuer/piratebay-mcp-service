实现一个种子搜索和下载的mcp服务，要求如下
1、搜索链接的示例为：https://thepiratebay10.xyz/search/A%20Knight%20of%20the%20Seven%20Kingdoms/2/99/0
参数说明
    A%20Knight%20of%20the%20Seven%20Kingdoms url编码后的搜索关键词
    2 代表第二页，默认从第一页开始
    其他参数保持默认即可
2、index.html即为搜索结果示例，实现一个搜索工具，传入关键词和页码后即可获取结构化的搜索结果

3、mcp服务提供下载工具，功能是帮助用户打开链接并下载
